Anmelden

Token erstellen: GitHub → Settings → Developer settings → Fine-grained tokens → „Generate new token" → nur dieses Repository auswählen → Berechtigung Contents: Read and write. Der Token bleibt ausschließlich in diesem Browser gespeichert.

Trainer hochladen

Die HTML-Datei wird in den Ordner trainer/ des Repositorys übernommen und erscheint danach automatisch im Portal. Gehören mehrere Dateien zusammen (z. B. eine .js-Datei), lade sie nacheinander hoch — nur die HTML-Datei braucht Titel und Beschreibung.
